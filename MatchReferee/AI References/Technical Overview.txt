MATCH REferee – Full Project Context Summary
================================================

1. Purpose & Mission
Match Referee is a UK-focused web platform that connects grassroots football coaches with qualified referees to eliminate the chaos of last-minute bookings, WhatsApp overload, and unreliable officiating.

Core Goal: Replace fragmented, manual referee coordination with a professional, automated, trust-based system — saving time, reducing stress, and improving match quality for referees, coaches, clubs, and leagues.

2. Target Users & Roles

├──────────────┼───────────────────────────────────────┼─────────────────────────────────────┤
│ Referees     │ WhatsApp spam, wasted travel, admin   │ Smart alerts, auto-scheduling       │
│ Coaches      │ No-show refs, poor quality            │ Verified ratings, local map         │
│ Clubs        │ Coach turnover, disorganisation       │ Central dashboard, branded page     │

3. Key Functionality (MVP – Launch Ready)
• Smart Matching (location, level, availability)
• Auto-Scheduling (one-click accept/decline)
• Mutual Ratings & Match History
• In-App Messaging (replace WhatsApp)
• Real-Time Calendar Sync
• Role-Based Access
• Admin Dashboard (clubs/leagues)
• Branded Club Page

No payment processing at launch — cash between coach/referee
